### Backend Devlopment

1. Create git repo
   git init

2. Add .gitignore file

3. Create .env file for environment variables & read data from .env with "dotenv" module
   npm i dotenv

4. Generate pacakge.json
   npm init -y

5. Create express app
   npm i express

6. Connect to Database
   -install mongoose npm i mongoose
   -connect to DB - using 'connect()' method in mongoose

7. Add middlewares (body parsers,error handlers)

8. Design Schemas and create models

9. Design REST APIs for all resources
